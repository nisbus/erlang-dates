Erlang-dates

Useful date_time functions

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Erlang Dates.

For now this only contains functions for creating date ranges for a given frequency.
Supported frequencies are:

  • years
  • months
  • quarters
  • weeks
  • days
  • hours
  • minutes
  • days
